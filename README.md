# TimeSync-flash
Web application to synchronize time to embedded devices using browser screen and light detector.

The data transfer is completed using a photodetector on the embedded device, in combination with flashing the screen on an internet connected device. Similar methodology is done in 
 * Timex Datalink (circa 1994)
 * Electric Imp Blinkup
 * [Clockie](https://github.com/mossblaser/Clockie) project by mossblaser
